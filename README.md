# Contact Management System (Python Flask)

## Features
- Add Contact
- View Contacts
- Update Contact
- Delete Contact
- Email Validation
- Duplicate Email Prevention

## Tech Stack
- Backend: Python Flask
- Frontend: HTML, Bootstrap
- Database: MySQL

## How to Run
1. Install dependencies:
   pip install flask mysql-connector-python

2. Run app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/
